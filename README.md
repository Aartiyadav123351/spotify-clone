# Spotify Clone

A simple Spotify-inspired music player built using HTML, CSS, and JavaScript. This project lets users browse a playlist, play/pause songs, skip tracks, and see live progress as the song plays.

## Features

- 🎵 Playlist of NCS (No Copyright Sounds) tracks
- ▶️ Play / Pause functionality
- ⏮️ ⏭️ Next and Previous track controls
- 📊 Live progress bar with seek support
- 🔁 Auto-plays next song when current one ends
- 🎨 Clean, responsive Spotify-style UI

## Tech Stack

- HTML5
- CSS3 (Flexbox layout)
- JavaScript (Vanilla JS, DOM manipulation)
- Font Awesome icons

## How to Run

1. Clone this repository
2. git clone https://github.com/Aartiyadav123351/spotify-clone.git
2. Open `index.html` in your browser (or use VS Code's Live Server extension)
3. Make sure your `songs/` folder contains the mp3 files referenced in `script.js`
